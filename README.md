# WGLScript toolchain

This extension provides support for the WGLScript programming language.

## Features

- [x] formatter
- [x] intellisense
	- [x] go to definition
	- [x] completions
	- [x] hover & signature help
	- [x] rename symbol (in ScriptModule or Project)
	- [x] find references (in ScriptModule or Project)
	- [x] diagnostics
